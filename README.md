# Peg Solitaire (Plain JS)

A simple Peg Solitaire game in plain HTML, CSS, and JavaScript. No libraries required.

## How to Play
1. Click a peg to select it.
2. Click an empty spot to jump over an adjacent peg.
3. Jumped peg is removed.
4. Goal: leave only one peg.

## Features
- Classic 7x7 board
- Valid move highlighting
- Board rotation buttons
- Move and peg count

## Usage

[Just play in your browser](https://robertplawski.github.io/peg-solitaire/), 
or clone this repo and open index.html
